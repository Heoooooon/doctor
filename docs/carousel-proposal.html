4 검색 노출(SEO) 고려

  • 키워드 글자는 이미지에 그려 넣지 않고, 사진 위에 실제 텍스트로 얹습니다. → 네이버·구글이 글자를 읽어 검색 노출에 유리합니다.
  • 모든 사진에 한국어 대체텍스트(alt)를 키워드 포함해 작성합니다. (예: "수원 네비게이션 임플란트 안내")
  • SEO 가이드의 핵심 키워드(수원 임플란트 · 수원 교정치과 · 수원 영통 치과 · 수원 소아치과)와 정렬됩니다.
  • 의료법 준수: 치료 결과 보장·최상급 표현은 사용하지 않습니다.

① "영통치과 / 매탄동치과" 연결 위치
두 키워드는 지역 키워드라 진료 페이지가 따로 없습니다. 기본안은 영통치과 → 병원소개, 매탄동치과 → 오시는 길입니다. 다른 페이지를 원하시면 알려주세요.
② "매탄동치과" 대신 "수원소아치과"로 교체할까요?
SEO 핵심 키워드에는 수원 소아치과가 포함되어 있습니다. 매탄동치과 대신 수원소아치과(→ 소아치과 /pediatric)를 넣으면 검색 타겟이 더 강해집니다. (현재 5개 그대로 갈지 / 교체할지)
③ 노출 위치
메인 페이지의 어느 위치에 배치할지(예: 첫 화면 아래 / 진료분야 섹션 근처). 지정 없으면 진료 안내 흐름에 맞춰 배치합니다.
④ 사진
위 "사용 이미지(안)"은 현재 보유 사진 기준입니다. 키워드별 전용 사진을 따로 주시면 교체해 드립니다.
