# 의료법 준수 가이드

> 홈페이지 콘텐츠 작성·수정 시 반드시 확인하세요.
> 의료광고는 의료법 제56조 및 의료광고심의위원회 기준을 따릅니다.

---

## 1. 필수 표기 사항

### 모든 페이지 공통
- 의료기관명: **서울이건치과**
- 대표 원장: **이재성**
- 사업자 번호: **770-17-01708**
- 주소·전화번호 표기
- **의료법에 의거, 치료 결과는 개인마다 다를 수 있습니다** 문구 (치료 설명 페이지)

### 개인정보 수집 (상담 신청 폼)
- 수집 항목 고지: 이름, 연락처
- 수집 목적 고지: 상담 예약 연락
- 보관 기간 고지
- 동의 체크박스 필수 (사전 체크 금지)

---

## 2. 금지 표현

의료법 제56조에 따라 아래 표현은 사용할 수 없습니다.

| 유형 | 금지 예시 | 대안 |
|------|----------|------|
| 치료 효과 보장 | "100% 성공", "반드시 나아집니다" | "많은 환자분들이 좋은 결과를 경험하셨습니다" |
| 최상급 표현 | "최고의 치과", "국내 최다 임플란트" | "오랜 경험의 전문의 직접 진료" |
| 타 병원 비교·비하 | "타 병원보다 저렴" | 단독 안내만 |
| 환자 후기 사용 | 환자 실명·사진 후기 | 사전 서면 동의 필수 |
| 미허가 효능 표현 | 의학적으로 검증 안 된 효과 주장 | 근거 있는 정보만 |
| 가격 비교 할인 강조 | "50% 할인", "이벤트 가격" | 심의 받은 경우 가능 |

---

## 3. 의료광고 심의

사전 심의가 필요한 광고 유형:
- 배너·팝업·SNS 광고
- 전후 사진(B/A) 사용 광고
- 가격 표기 광고

**심의 기관**: 대한의사협회 의료광고심의위원회

홈페이지 콘텐츠(게시판 설명 글)는 정보 제공 목적이면 심의 제외 가능하나, 광고성 문구 추가 시 심의 필요.

---

## 4. 의료진 정보 표기 규칙

- 전문의 표기: 반드시 해당 전문과목 명시
  - 유수현 원장 → "교정과 전문의"
  - 백설아 원장 → "소아치과 전문의"
- 학력·경력 표기: 사실에 근거, 과장 금지
- "서울대 출신" 표현: 학력 사실이므로 표기 가능 (졸업 증빙 유지)

---

## 5. 개인정보 처리

| 항목 | 기준 |
|------|------|
| 수집 항목 | 최소화 원칙 (이름, 연락처만) |
| 보관 기간 | 상담 완료 후 6개월 또는 동의 철회 시 즉시 삭제 |
| 제3자 제공 | 원칙적 금지, 필요 시 별도 동의 |
| 관리자 접근 | 로그인 인증 필수 (`/admin` 경로) |

---

## 6. 치료 전후 사진(B/A) 사용

- 환자 사전 서면 동의 필수
- 신원 특정 불가한 형태로만 사용 (얼굴 마스킹 또는 구강 내 사진만)
- 과장 편집 금지 (보정·합성 불가)
- 의료광고심의 통과 후 사용 권장
